Notepad

Welcome to Notepad. Notepad is a web based text editor which uses localstorage to save its content. It can load text content from text files and also save the written contents as text files. It provides a beautiful UI and a clean Interface to do the tasks. You can handle multiple files at once. It autosaves your documents every 30 seconds. It can export your information in text format.
